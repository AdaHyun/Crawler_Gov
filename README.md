# NHC & CDC Public Health Document Crawler

本项目用于爬取和整理中国公共卫生相关官方网站中的政策文件、疫情信息、科普文章、监测报告等公开数据。当前覆盖的数据来源包括：

- 中国疾病预防控制中心
- 中国疾控中心慢病中心
- 国家疾病预防控制局
- 国家卫生健康委员会

项目主要输出结构化的 JSONL 文档数据，并保留爬虫运行日志与附件统计信息，便于后续进行公共卫生文本分析、政策文件整理、知识库构建或大语言模型数据集构建。

------

## 1. 项目目标

本项目的主要目标是从公共卫生相关官方网站中自动化采集公开文档数据，包括：

1. 政策法规类文件
2. 疾病预防控制标准
3. 疫情信息
4. 传染病监测报告
5. 疾控科普文章
6. 健康生活方式科普内容
7. 统计数据与工作通知
8. 学术动态、科研平台、国际合作等相关信息

采集后的数据统一保存为结构化文件，便于后续清洗、检索、分析和建模使用。

## 4. 项目目录结

项目的主要目录结构如下：

```text
nhc_crawler/
├── config/
│   └── sites.json                         # 爬取站点与栏目配置
├── data/
│   ├── output/                            # 结构化输出数据
│   ├── logs/                              # 爬虫运行日志
│   └── statistic/                         # 数据统计结果
├── scripts/
│   ├── export_attachment_stats.py         # 统计附件下载情况
│   └── stats_by_institution_project.py    # 按机构、栏目及分类字段统计数据情况
├── src/
│   ├── main.py                            # 项目主入口
│   └── parser/                            # 不同网站的详情页解析模块
│       ├── chinacdc_detail.py
│       ├── ncncd_detail.py
│       ├── ndcpa_detail.py
│       └── nhc_detail.py
├── .gitignore
└── README.md
```

## 7. 数据格式说明

输出数据采用 JSONL 格式，即每一行是一个独立的 JSON 对象。该格式适合大规模文本数据存储，也便于后续使用 Python、Pandas、数据库或大语言模型训练管道进行处理。

## 9. 使用场景

本项目采集的数据可用于：

1. 公共卫生政策文本分析
2. 疫情信息与传染病报告整理
3. 疾病预防控制知识库构建
4. 中文公共卫生语料库建设
5. 政策法规与标准文件检索
6. 大语言模型领域数据构建
7. 公共卫生问答系统或 RAG 系统开发

------

## 10. 注意事项

1. 本项目采集的数据均来自公开网站页面，仅用于学习、研究和数据分析。
2. 数据内容版权归原发布机构所有。
3. 若用于公开发布、模型训练或产品系统，应遵守对应网站的使用规范与数据合规要求。
4. 大型附件、网页原始 HTML、图片等文件默认不上传至 GitHub，以避免仓库体积过大。
5. 若需要完整附件数据，建议单独使用云存储、Release 文件或其他数据托管方式管理。
